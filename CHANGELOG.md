# Changelog - Gourmet Express 📜

Todas las actualizaciones y cambios notables de este proyecto se documentarán en este archivo.

## [1.0.0] - 2026-05-10
### Añadido
- Estructura base del proyecto con HTML, CSS y JS.
- Menú interactivo con 9 platos típicos colombianos.
- Sistema de carrito de compras funcional.
- Módulo de seguimiento de pedidos para meseros.
- Generación de comprobantes en PDF.
- Persistencia de datos con `localStorage`.

### Mejorado
- Rediseño total de la interfaz (UI Overhaul) con estilo Premium y Glassmorphism.
- Estabilización de imágenes utilizando el CDN de Pexels.
- Optimización del proceso de checkout para ingreso manual de clientes.
- Mejora de la responsividad para dispositivos móviles y tablets.

### Corregido
- Error de carga de imágenes por bloqueo de red (Migración de Unsplash a Pexels).
- Error en el cursor de los botones de navegación (Añadido `cursor: pointer`).
- Error de visualización en la cuadrícula del menú (Ajustado a 3 columnas).
